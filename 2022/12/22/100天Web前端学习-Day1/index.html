Day 1

HTML 概述

HTML 全称是 HyperText Mark-up Languae 是一种超文本标记性语言。

常见用于给 Internet 上的资源进行链接为一个逻辑体，通过标签把网络上的文档进行格式统一。

HTML 没有逻辑运算的规则，所以并不能用来进行编程或程序编写。

HTML 是通过它的内置标签语言来进行对象的编辑、排版、格式等内容。

HTML 常见功能

HTML 常见用于 Web 前端开发。
展示在线的文档，其中包含了标题，文本，图片，表格等内容。
在线编辑文本内容，电子表格，视频剪辑等。
通过超链接搜索在线的信息。

HTML 文本结构

<DOCTYPE html> 用于声明文档类型是 HTML 文件

HTML 一共分为三个部分，这里是第一个标签，涵盖了所有页面内容，

<html lang="zh-CN"> 声明本文档中使用的主要语言

第二部分，head 是指头部，主要用于配置 HTML 文件信息，例如引入外部文件

<head>

<meta charset="UTF-8"> 文档编码设置

<meta http-equiv="X-UA-Compatible" content="IE=edge"> 兼容 IE 浏览器设置

<meta name="viewport" content="width=device-width, initial-scale=1.0"> 网页自动适配缩放

<title>这里写标题</title> 网页标题

</head>

第三部分，body 是指身体，页面的内容将会在在这里面

<body>

<p>页面内容</p> p 标签是一个段落标签，常用于书写文本内容

</body>

  • <div> </div>

    div 标签常常用于建立一个新的内容区块，分层，无特殊含义

  • <h1> </h1>

    一级标题

  • <h2> </h2>

    二级标题

  • <h3> </h3>

    三级标题

  • <h4> </h4>

    五级标题

  • <h6> </h6>

    六级标题

  • <p> </p>

    p 标签用于添加文本段落

  • <br>

    换行

  • <hr> 产生一条分割线,如下 ↓↓↓↓

  • <a>

    超链接标签，可以在页面上添加一个超链接，可以是文本，图片等

  • <img>

    添加图像

  • <span>

    和 div 标签一样，同样能建立一个区块，但区别是 div 是占据一整行内容，而 span 是紧凑的，没有换行。

  • <ul>

    无序列表 如下 ↓↓↓↓
